# Neo-Bank Dashboard

A crypto-friendly banking dashboard for startups.

## Features

- **Treasury Dashboard**: View crypto and fiat balances and recent transactions.
- **Bills & Invoices**: Manage and create invoices.
- **Budgets & Alerts**: Track spending and configure notifications.
- **Agent Chat**: Interact with an AI financial assistant.

## Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS + shadcn/ui
- **Language**: TypeScript
- **Web3**: ethers.js

## Project Structure

This project uses a feature-based folder structure inside `src/`. Please refer to the `README.md` files in `src/components`, `src/lib`, and `src/hooks` for more details.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) with your browser.
